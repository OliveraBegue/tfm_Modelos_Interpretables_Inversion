# Bloque 1 — Preparación de datos

**Notebook:** `Bloque_0_v6.ipynb`

Contiene:
- Ingesta de datos del S&P 500 y variables macro/técnicas/sectoriales (2004–2024).
- Construcción de etiquetas mediante Triple Barrier Labeling.
- Ingeniería de variables (tendencia, momento, volatilidad, estrés de mercado, ciclo económico, rotación sectorial).
- Eliminación de multicolinealidad (correlación + VIF).

**Output:** dataset limpio y etiquetado, base para las iteraciones 1 y 2.
